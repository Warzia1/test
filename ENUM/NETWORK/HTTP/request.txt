GET Request

This is used for getting information from a web server.

ex:	 (Make a GET request to /blog and using the gear icon set the id parameter to 1 in the URL fiel)
	GET HTTP/1.1
	GET /blog?id=1 HTTP/1.1


POST Request

This is used for submitting data to the web server and potentially creating new records

ex:	(POST the username of thm and a password of letmein to /login)
	POST /login HTTP/1.1
	Host: tryhackme.com
	User-Agent: Mozilla/5.0 Firefox/87.0
	Content-Length: 33

	username=thm&password=letmein


PUT Request

This is used for submitting data to a web server to update information

ex:	(Make a PUT request to /user/2 with the username parameter set to admin)
	PUT /user/2 HTTP/1.1
	Host: tryhackme.com
	User-Agent: Mozilla/5.0 Firefox/87.0
	Content-Length: 14

	username=admin


DELETE Request

This is used for deleting information/records from a web server.

ex:
	DELETE /user/1 HTTP/1.1 (Make a DELETE request to /user/1)
