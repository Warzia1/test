Script in Lua more option on : https://nmap.org/book/nse-usage.html

Scan  complet « rapide » 
Nmap -A -r [IP]

scan vuln :
nmap -sV --script vuln [ip]

OPTIONS :

pour mettre dans un fichier :
-oN nmap-$ip.out 
-Pn = Disable host discovery and scan for open ports
-sC = Scan with the default Nmap scripts
-sU = UDP port scan
-sS = TCP SYN port scan

sneaaky scan (null flag TCP): 
-sN 

FIRST SCAN : 
nmap -A -T4 [ip]

SECOND :
nmap -p- -Pn -T4 [ip]
