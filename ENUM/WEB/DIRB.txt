Dirb est une commande Linux qui permet de scanner un site Web pour détecter 
des fichiers et des répertoires cachés. Elle fonctionne en envoyant des requêtes 
HTTP à un site Web et en analysant les réponses pour identifier l'existence d'un 
fichier ou d'un répertoire.

La syntaxe de la commande dirb est la suivante :

dirb [options] [site Web]
Options

La commande dirb prend en charge un certain nombre d'options, qui permettent 
de personnaliser le scan.

-r : Effectue un scan récursif du site Web.
-w : Utilise un fichier de mots de passe pour les requêtes HTTP.
-t : Définit le nombre de threads à utiliser pour le scan.
-u : Définit l'utilisateur agent à utiliser pour les requêtes HTTP.
-x : Définit les extensions de fichiers à rechercher.

Exemples

Pour scanner un site Web en utilisant un fichier de mots de passe :

dirb -r -w /path/to/wordlist.txt http://example.com
Pour scanner un site Web en utilisant un nombre spécifique de threads :
dirb -r -t 10 http://example.com
Pour scanner un site Web en utilisant un utilisateur agent spécifique :

dirb -r -u "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/97.0.4692.99 Safari/537.36" http://example.com

Pour scanner un site Web en recherchant uniquement des fichiers avec les 
extensions .php et .asp :

dirb -r -x php,asp http://example.com
